# Tesis Módulo Web FACE 🏫 Aplicación web desarrollada para la Facultad de Ciencias Empresariales (FACE) de la Universidad del Bío-Bío, que permite gestionar avisos clasificados y promover actividades grupales de manera eficiente. --- ## Características Principales - **Gestión de Avisos Clasificados**: Publicación y gestión de avisos relacionados con compra/venta de artículos y servicios. - **Promoción de Actividades Grupales**: Detalles y horarios de actividades organizadas dentro de la facultad. --- ## Tecnologías Utilizadas - **Servidor Web**: Apache (contenedorizado con Docker) - **Lenguaje**: PHP - **Base de Datos**: MySQL (contenedorizado con Docker) - **Contenedores**: Docker, Docker Compose --- ## Versiones - **PHP**: 8.2 - **MySQL**: 5.7 - **Docker**: 20.10 o superior - **Docker Compose**: 1.27 o superior > ⚠️ **Nota**: Usar estas versiones garantiza la compatibilidad y estabilidad del proyecto. --- ## Estructura del Proyecto ```plaintext SITE/ ├── wp-admin/ ├── wp-content/ │ ├── languages/ │ ├── plugins/ │ │ ├── avisos-clasificados/ │ │ ├── contact-form-7/ │ │ ├── gtranslate/ │ │ ├── ... (otros plugins) │ └── themes/ ├── wp-includes/ ├── .htaccess ├── Dockerfile ├── docker-compose.yml └── wp-config.php
